# Key Performance Indicators (KPIs)

These are the 5 KPIs we use to measure the success
of this project.

## KPI 1 — Login Success Rate
- **What it measures:** How many users log in successfully
- **Target:** 90% or higher success rate
- **How to measure:** Count successful logins vs failed logins

## KPI 2 — Input Validation Errors
- **What it measures:** How often users submit empty or wrong input
- **Target:** Less than 10% error rate
- **How to measure:** Count how many times validation warning appears

## KPI 3 — Page Load Time
- **What it measures:** How fast the login page loads
- **Target:** Under 3 seconds
- **How to measure:** Open page and time how long it takes to fully load

## KPI 4 — Authentication Failures
- **What it measures:** How many wrong password attempts happen
- **Target:** Less than 5 failed attempts per session
- **How to measure:** Count incorrect login attempts

## KPI 5 — User Satisfaction
- **What it measures:** Whether users find the system easy to use
- **Target:** 8 out of 10 or higher
- **How to measure:** Ask classmates to rate the system from 1 to 10
